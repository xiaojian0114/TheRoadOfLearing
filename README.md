# 📦 全栈学习项目合集
本仓库汇总了个人全栈技术学习过程中的各类项目与练习代码，按技术栈分类归档，方便查阅与回顾。

---

## 📁 项目目录说明
| 目录 | 技术栈 | 项目说明 |
| :--- | :--- | :--- |
| **HarStudy** | 鸿蒙 (HarmonyOS) | 鸿蒙开发学习代码，包含ArkTS、ArkUI等相关实践 |
| **Harmony** | 鸿蒙 (HarmonyOS) | 鸿蒙应用开发进阶练习与Demo项目 |
| **Microservice_Learning** | Spring Cloud / 微服务 | 微服务架构学习代码，涵盖服务注册发现、网关、熔断等核心组件 |
| **React_Learing** | React.js | React前端框架学习笔记与实战练习 |
| **front-study** | Vue 3 | Vue 3前端开发学习项目，包含Composition API、Pinia等实践 |
| **my-express-service** | Express.js (Node.js) | Node.js Express后端服务学习代码，接口开发与服务搭建实践 |
| **react-app** | React.js | 基于React开发的完整前端应用项目 |
| **spring-boot** | Spring Boot 3 | Spring Boot 3后端开发学习代码，包含Spring生态、数据库操作等实践 |
